GRA 451 - Advanced Multiplatform Design

The line between a website designer and a website developer continues to blur. After learning about responsive web design in previous coursework, you will add to this previous knowledge to develop an interactive, responsive website using appropriate tools and languages. A website designer needs to recognize the impact design decisions have on the development of the website. The final assessment in this course is to design and develop an interactive, responsive website that meets the provided project parameters. You will create a wireframe and mock-ups before developing more advanced interactive features using JavaScript. The final assessment will provide you the opportunity to demonstrate your ability to develop and design an interactive, responsive website.

Design Process

I had fun coming up with a design for the Java Been company. I love neutral colors, so I chose different shades of brown for my design. I thought these colors would be great to go along with a coffee company. I decided to add some design elements to the mockup to really show what I envision the website design to be. Again, I have these swirly design elements incorporated within each page design so that it would make each page look “creamy.” The wireframes I created, allowed me to plan out the ideas I had in my head for each page. It also helped me map out how I would like the layout of the mobile version to be to correlate with the desktop version. To incorporate project specifications, like the number of pages, and the use of the style guide, I had to make sure I design the website the way I wanted to but also in a way that I’m paying attention to the specifications. I also had to take into account how each page would look like on a desktop and mobile site. Knowing that with a mobile device, pages get smaller, I had to think of how to make the design cohesive so that if a user does switch to a mobile device, the words won’t be huge, or the design elements won’t be fixed at the desktop size. Another way I integrated specifications of the project was incorporating the customer demographic from the style guide into my design choices, incorporating a responsive navigation bar that is consistent on every page, and incorporating different programming languages to create functionality performance to the best of my ability. I used a variety of tools to help me achieve the final design. To create the wireframes and mockups I used a combination of two editing programs: Adobe Illustrator and Photoshop. To bring the designs to life I used Visual Studio Code and GitHub with the use of HTML5, CSS3, and JavaScript.
