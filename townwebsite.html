Assignment

Create a functioning and compliant website. To design a functioning and compliant website, you will use HTML with an external CSS file that defines the style and formatting. The standard HTML tags and techniques that are presented in this course will be your building blocks for the design or content on your website. HTML is the hierarchical structure of information for a webpage design, whereas CSS controls how the HTML looks (colors, position, and size) and how it's presented in a browser. Your website will also demonstrate your understanding of the foundations of file optimization and management, which are imperative to web design and development.

Reflection

Reflecting on my journey through this web design class, I'm struck by the profound transformation I've experienced. Initially, I approached the course with a mix of trepidation and skepticism, worrying that it might be the first class I'd ever fail. Yet, here I am, not only having succeeded but also filled with pride over what I've achieved. The project started with a deceptively simple task that masked the challenges ahead. Despite my early anxieties, especially about coding—a skill I hoped to avoid—the structured modules gradually built my confidence. Sketching wireframes turned a corner for me, transforming abstract ideas into tangible plans. Among the technical challenges, alignment and understanding webpage layout were my biggest obstacles, alongside mastering padding and troubleshooting a persistent navbar issue. The latter proved particularly vexing; the dropdown menus wouldn't behave as intended. It took several painstaking hours of code adjustment before I pinpointed the margin settings as the culprit. In developing the "TOWN Restaurant" website, I leaned heavily into my developing skills in graphic design and coding at the time. My deliberate choices in imagery, typography, and color contrast were aimed at creating an interface that wasn't just visually appealing but also user-friendly. Employing Adobe Photoshop allowed me to manipulate images with precision, while Visual Studio Code brought my designs to life in HTML and CSS. This project underscored the importance of layering, blending modes, and UX design principles—techniques that ensure a website not only looks good but functions well for its intended audience.
